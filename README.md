# Baseerah

مشروع بصيرة - واجهة عربية لتطبيق FinTech بسيط.

## تشغيل محليًا

1. انسخ `.env.example` إلى `.env.local`
2. اضف قيم Supabase:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. شغل:
   ```bash
   npx vercel dev
   ```

## نشر على GitHub / Vercel

1. ارفع المشروع إلى GitHub
2. استورد المشروع في Vercel
3. اضف متغيرات البيئة نفسها في Vercel
